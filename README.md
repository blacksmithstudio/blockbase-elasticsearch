# Driver Elasticsearch for Blockbase
Compatible with Blockbase Framework

### Version
0.0.3 alpha

### How to install ?
```shell
$ npm i --save blockbase-elasticsearch
```

Then add to your config/{env}.yml the following instructions depending of your system
```yml
dbms : elasticsearch
elasticsearch :
    host : localhost:9200
    index : myindex
```
